Joylashuv va static qiymat

Qiymatlar

  • CSS joylashuvi veb-sahifa ichidagi elementlarning joylashishini boshqarish haqida.
  • CSS joylashuvi bilan siz oddiy hujjat oqimini bekor qilishingiz mumkin.

Barcha HTML elementlari sukut bo'yicha statik holatda joylashtiriladi. Statik joylashtirilgan elementlarga yuqori, pastki, chap va o'ng xususiyatlar ta'sir qilmaydi. bilan elementi position: static;har doim sahifaning normal oqimiga muvofiq joylashtiriladi:

  • static- Bu standart holatda. Element oddiy hujjat oqimiga muvofiq joylashtirilgan
  • relative- Element hujjat oqimidagi normal holatiga nisbatan joylashtirilgan
  • fixed- Element ko'rish maydoniga nisbatan joylashtirilgan/li>
  • absolute- Element eng yaqin joylashgan ajdodga nisbatan joylashtirilgan
  • sticky- Element aylantirish holatiga qarab nisbiy va belgilangan holat o'rtasida almashinadi
